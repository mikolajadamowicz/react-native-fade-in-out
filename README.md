# react-native-fade-in-out

Reanimated fade in / fade out animation

## Installation

```sh
npm install react-native-fade-in-out
```

## Usage

```js
import FadeInOut from "react-native-fade-in-out";

// ...

const result = await FadeInOut.multiply(3, 7);
```
